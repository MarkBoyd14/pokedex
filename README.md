# Pokedex App

Pokedex app that gets data from PokeAPI (a RESTful Pokemon API)

- Created with React
- Displays a list of 18 Pokemon per page
- Paginate using "Prev" and "Next" buttons
- Displays Pokemon sprites, type(s), weight, height, and abilities
